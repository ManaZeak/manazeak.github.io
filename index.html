Fed up getting commercials while listening to your music online ?

You are not alone!

ManaZeak is a self-hostable web application that is based on your musical library. Share it with friends, with no restrictions and enjoy together your exquisite array of musical genius!

ManaZeak can be deployed in a few minutes thanks to docker. For further information on how to get started with it, refer to the repository wiki. Currently in beta version, ManaZeak features a music player, a library manager, a implementation of the moodbar visualization, a user system and two different views (so far).

History

In September 2017, ManaZeak was born under a final study project, at ESIEA Paris (top french engineering school). By the end of this project in February 2018, we decided to make it a real open source project so ManaZeak could grow further. Several people where involved in the journey and helped us out on miscelaneous subjects, as you can see on the following cards.

ManaZeak was getting more and more features, but despite our efforts, we had to Ctrl + A + Suppr all the frontend codebase as summer 2018 was arising: code architecture was not made to handle properly all of our ideas. We almost restarted from scratch, with a better vision of how should we do our logic in there, and here we are today: not as quite features as we had in the past, but far more efficient and with a much cleaner UI. Now that the frontend code base has been cleansed, it's backend refactor time for the same reasons! We expect to get the same level of features as for the ManaZeak alpha version when we will reach the Beta 0.3 version : Make Mzk Great Again. Keep on touch to see what's coming!
