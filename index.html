Fed up with ads when listening to music online ?

You are not alone!

ManaZeak is a self-hostable web application that is based on your musical library.
Share it with friends, with no restrictions and enjoy together your exquisite array of musical genius!

It features a solid user system, based on invitation codes from already registered user so you have control over who has access to your instance. Users can then browse the library with several views (per artists, albums, years, genre, labels etc.). It also implement moodbars to have a great visualization of each track's style.

Based on a clean, well-tagged library, this software offers several views that takes the most of those data to give the best experience. For more information, check the audio naming convention to understand how to prepare your files.

This project is still under development, and a lot of essential features are still on their way to this platform ; audio visualizers, playlists, smart suggestions, advanced sorting, musicology, online editing etc., so keep on touch!

About ManaZeak

        

In September 2017, ManaZeak was born as a graduation project, at ESIEA Paris (a top French engineering school). By the end of this academic project in February 2018, we decided to make it a real open source project so ManaZeak could grow further. Several people were involved in the journey and helped us out on miscellaneous subjects.

Two years later, the projects architecture was a true limitation to any new feature. Maintainers then decided to switch the backend from Django to Spring boot, to answer those scalability issues. This rework is currently in progress and promises a few great improvements, while stepping up the codebase bedrock.

Because we want to provide the best software we can, we use several tools to perform routines that ensure the app meets our quality standards. Use the following cards to check the different services we use as our continuous integration workflow. Also, if you want to know what's coming next in ManaZeak, check out the YouTrack/Trello dashboards that lists all the upcoming features.


To help you go through the installation process, you can consult the repository's wiki to have a better understanding on how to manage a ManaZeak instance. If you wish to learn more about how ManaZeak is working, refer to the online documentation, that explains both the frontend and the backend code.

This project is open source, released with a GPL-3.0 license. It represents our vision of an online listening platform, but we might be missing some great ideas! Feel free to reach us using mail/DM, or open an issue with your feature request so we can fulfil it.
